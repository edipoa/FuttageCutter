FUTTAGE CUTTER v1.0.0
===================

Aplicativo para cortar vídeos usando FFmpeg.

REQUISITOS
- Windows 1011
- .NET 8.0 Runtime (será verificado durante a instalação)

COMO USAR
1. Selecione o arquivo de vídeo de entrada
2. Defina o tempo de início e fim do corte
3. Escolha onde salvar o arquivo cortado
4. Clique em Cortar Vídeo
